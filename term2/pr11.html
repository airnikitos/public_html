В jalview-проекте(кликабельно для скачивания) я, при помощу функции calculate -> Tree -> average distance, разделил последовательности на группы (из них две получились превалирующими по размеру, зеленая и синяя). Это разделение можно увидеть, посмотрев на раскраску позиций по консервативности. У обоих групп есть много общих высококонсервативных участков, но у каждой из них так же есть участки, не характерные другой.
Например, позиции 40-42 и 88-89 являются высококонсервативными, для зеленой группы, а для синей нет. Пример общих высококонсервативных участков - позиции 71-73
