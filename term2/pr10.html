Задание 3

По новому запросу (добавился фильтр по организмам : viruses) было найдено 54 последовательности. Из теоремы Карлина можно сделать вывод ( так как K, m, lambda и S ,Будут неизменными), что доля вирусных белков можно оценить зная только E-value для двух поисковых запросов(один с фильтром по организму). Так что: n(viruses)/n=E-value(viruses)/E-value. Было взято значение для одинаковой находки - белка POC6E7.1.

Вычисление - n(viruses)/n = 7e-24/2e-22 = 0.035. Из этого следует, что доля вирусных в белков в Swiss-Prot равняется 3.5%.
